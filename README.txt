READ ME

Pour utiliser l'application, il faut effectuer les instructions suivantes : 
1. Utiliser un des navigateurs suivants : Google Chrome, Mozilla Firefox ou Opera, et s'assurer qu'ils sont à jour. WebRTC n'est pour le moment supporté que par les récentes versions de ces navigateurs.
2. Installer un serveur web sur sa machine (wamp, lamp, mamp, tomcat...).
3. Créer un nouveau site web sur votre serveur web.
4. Télécharger le repertoire et mettre les fichiers décompressés dans le repertoire de votre nouveau site web.
5. Télécharger Etherpad project à l'adresse https://github.com/ether/etherpad-lite
6. Contacter l'entreprise Bistri (www.bistri.com) afin d'obtenir des clés d'API valides et des crédits d'appel. Nous rappellons que ce code utilise l'API et les serveurs de signalisation de la société Bistri.
7. Dans le fichier conferencejs.js, remplacer les clés de l'API par vos propres clés obtenues à l'étape précédente.
8. Créer un nom de domaine virtuel comme celui indiqué par Bistri.
9. Aller sur la page index en utilisant le nom de domaine créé précédemment.
10. Cliquez sur le bouton "Créer une nouvelle conférence". Vous êtes dans la conference room. Partagez le lien avec vos amis afin qu'ils puissent vous rejoindre.
11. Enjoy ! :) 




To use the application , you have to perform the following steps:
1 . Use Google Chrome, Mozilla Firefox or Opera and make sure they are up to date. WebRTC does for now only support recent versions of these browsers.
2 . Install a web server on your machine ( wamp , lamp , mamp , tomcat ... ).
3 . Create a new web site on your web server.
4 . Download and save the unzipped directory in the directory of your new website files.
5 . Donwload Etherpad project on: https://github.com/ether/etherpad-lite
6 . Contact the company Bistri ( www.bistri.com ) to obtain valid API key and call credits. We remind that this code uses the API and servers signaling Bistri society.
7 . In conferencejs.js file, replace the API key with your own key obtained in the previous step.
8 . Create a virtual domain name as that specified by Bistri.
9 . Go to index page using the domain you created earlier.
10 . Click the "Launch a new visioconference" button. You are now in the conference room. Share the link with your friends so they can join you in the conference room.
11 . Enjoy ! :)
